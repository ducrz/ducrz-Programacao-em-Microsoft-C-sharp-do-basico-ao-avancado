29. Algoritmos de ordenação.


Algoritmo é um processo de resolução de uma determinada tarefa para a qual ele foi designado.

Uma "fórmula" para resolver um problema.



Um algoritmo de ordenação têm como objetivo realizar a ordenação de uma lista de valores.


BUBBLE SORT - Flutua o elemento maior até o fim da lista

É um algoritmo de ordenação simples que tem como característica percorrer a lista várias vezes e a cada passagem
faz "flutuar" para o topo o maior elemento da sequência.


A lógica utilizada
Temos três valores que não estão em ordem(9,5,7).
O 9 é o maior e neste caso seria o último algarismo dos três.

Para efetuar a ordenação, temos que perguntar se o algarismo da direita é maior que o da
esquerda e dependendo da resposta é feita uma troca de posição no próprio vetor, utilizando
uma variável auxiliar, assim será trocado o algarismo que for maior com o menor,
colocando o maior sempre para a direita até achar um númério que não seja menor que ele.

30. Algoritmo de busca binária.


Busca Binária

A busca binária é um algoritmo de busca que segue o paradigma de divisão e conquista.

Partindo do pressuposto de que o conjunto de elementos está ordenado, são executadas
diversas divisões do espaço de busca, restringindo o possível local no qual o elemento
buscado está posicionado.



Pseudocódigo

definição do vetor contendo números em ordem
definição variáveis sup, inf, meio
int = -1;
sup = vetor.Length

Enquanto (inf < sup -1)
	meio = (inf+sup)/2;
	Se(vetor[meio] < valorBuscado)
		inf = meio
	Senão
		sup = meio;
